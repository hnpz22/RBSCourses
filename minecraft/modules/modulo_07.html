Del Visual al Texto — PSeInt

¿Qué es el pseudocódigo? · Variables declaradas en texto
💬 Si Scratch usa bloques de colores para programar, ¿cómo crees que se ve exactamente ese mismo programa escrito en texto puro? ¿Crees que es más fácil o más difícil de leer?
▶ PARTE 1 — CONCEPTUALIZACIÓN 60 MIN
  • Introducción a PSeInt: Descargar o abrir. Estructura obligatoria: "Algoritmo nombre / Inicio / [instrucciones] / Fin". El pseudocódigo es un puente entre el lenguaje humano y el lenguaje de máquina.
  • Comparación Scratch → PSeInt en vivo: El docente muestra el mismo programa en ambas herramientas. "Repetir 10 veces" → "Para i←1 Hasta 10 Con Paso 1 Hacer". "Si toca borde" → "Si color = rojo Entonces Escribir '¡Peligro!'"
  • Variables en PSeInt: Declaración implícita. Asignación con el operador ←. Tipos: Real (número decimal), Entero, Cadena (texto), Logico (verdadero o falso).
  • Primer programa completo: Pedir el nombre del usuario con Leer. Calcular la edad en meses (edad * 12). Mostrar el resultado con Escribir usando una cadena formateada.
FÁCIL: Programa que saluda usando el nombre del usuario MEDIO: Calculadora de edad exacta en días y meses DIFÍCIL: Conversor completo de temperatura C → F → K con fórmulas
▶ PARTE 2 — MINECRAFT LAB 60 MIN
  • Introducción al Agent: Robot programable interno de Minecraft Education. Comandos disponibles: mover (adelante, atrás, izquierda, derecha), girar, colocar bloque, recoger bloque.
  • Primer programa del Agent: Caminar en línea recta 10 pasos colocando un bloque en cada posición. Verificar visualmente que el resultado es correcto.
C2

Ciclos y Condicionales en PSeInt

Traducir el minijuego de Scratch a pseudocódigo completo
💬 Ya sabes exactamente cómo funciona un loop en Scratch con bloques de colores. ¿Cómo crees que se escribe "repetir 10 veces" cuando no hay bloques, solo texto?
▶ PARTE 1 — CONCEPTUALIZACIÓN 60 MIN
  • Ciclo Para en PSeInt: "Para i←1 Hasta 10 Con Paso 1 Hacer / [instrucciones] / FinPara". La variable i toma los valores del 1 al 10 automáticamente en cada iteración.
  • Ciclo Mientras en PSeInt: "Mientras [condicion] Hacer / [instrucciones] / FinMientras". Se repite mientras la condición sea verdadera. Advertencia: loop infinito si la condición nunca se vuelve falsa.
  • Si-Entonces-SiNo anidado: Condicionales dentro de otros condicionales para decisiones complejas. La indentación visual es clave para leer el código correctamente.
  • Ejercicio principal de la clase: Traducir el minijuego de Scratch al pseudocódigo. El docente guía con preguntas: "¿Cuál era el loop principal? ¿Cuál era la condición de victoria?"
FÁCIL: Ciclo que imprime los números del 1 al 10 MEDIO: Programa que determina si un número es par o impar DIFÍCIL: Pseudocódigo completo del minijuego con score y condición de victoria
▶ PARTE 2 — MINECRAFT LAB 60 MIN
  • Agent con ciclo: Construir una pared larga usando Para: colocar bloque → mover → repetir N veces. Cambiar el valor de N para observar el efecto en la longitud de la pared.
  • Agent con condicional: Si hay un bloque adelante → girar 90°. Si no → avanzar y colocar bloque. El Agent navega un espacio sin chocar.
  • Inicio estación de transporte: El Agent coloca los rieles del sistema de transporte. El estudiante construye la estructura de la estación alrededor de los rieles.
C3

Funciones y Agent Constructor Avanzado

SubAlgoritmos reutilizables · Automatizar construcción compleja
💬 Si tuvieras un robot constructor físico real que siguiera tus instrucciones exactas sin error, ¿qué le ordenarías construir primero en tu ciudad?
▶ PARTE 1 — CONCEPTUALIZACIÓN 60 MIN
  • ¿Qué es un SubAlgoritmo? Un bloque de código con nombre propio que realiza una tarea específica y puede reutilizarse en cualquier punto del programa. Evita repetir el mismo código múltiples veces.
  • SubAlgoritmo en PSeInt: "SubAlgoritmo nombre(parametros) / [instrucciones] / FinSubAlgoritmo". Se invoca desde el algoritmo principal usando su nombre.
  • ¿Por qué son fundamentales? Si necesito construir 4 paredes, creo "construirPared(longitud)" y la llamo 4 veces con distintos valores. El código queda limpio, corto y fácil de mantener.
  • Ejercicio: SubAlgoritmo que calcula el área y perímetro de un rectángulo. Llamarlo 3 veces desde el algoritmo principal con valores distintos.
▶ PARTE 2 — MINECRAFT LAB 60 MIN
  • Agent construye una habitación completa: Programar las 4 paredes usando ciclos anidados (Para dentro de Para). Absolutamente prohibido colocar bloques manualmente.
  • Actividad física — Red de transporte humana: 5 estudiantes son "estaciones" fijas. Los demás son "trenes". El "programador" da instrucciones verbales exactas de movimiento. ¿Qué instrucciones necesitaba el sistema?
  • Red de rieles en Minecraft: Conectar 3 zonas distintas de la ciudad con rieles funcionales. Añadir estaciones con señalización clara.
